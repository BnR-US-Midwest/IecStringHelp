IecStringReplace

Find in source and replace in destination.

The function returns a status code, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN source STRING Source string
IN find STRING Find string
IN replace STRING Replace string

Possible Errors

Constant Description Remedy
IECSTRING_ERROR_NULL Destination, source, find, or replace strings are null (the memory address is zero) All strings must have a valid memory address
IECSTRING_ERROR_SIZE_ZERO Size is zero Size must be non-zero
IECSTRING_ERROR_OVERLAP The size of destination overlaps with the length of source, find, or replace Destination's memory location must differ from source's and find's and replace's
IECSTRING_WARNING_TRUNCATE The length of the resulting string exceeds the size of destination Increase the size of destination
