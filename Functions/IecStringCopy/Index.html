IecStringCopy

Copy source to destination up to size of destination or source length.

The function returns a status code, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN source STRING Source string

Possible Errors

Constant Description Remedy
IECSTRING_ERROR_NULL Destination and/or source string is null (the memory address is zero) Destination and source must have a valid memory address
IECSTRING_ERROR_SIZE_ZERO Size is zero Size must be non-zero
IECSTRING_ERROR_OVERLAP The size of destination overlaps with the length of source Destination's memory location must differ from source's
IECSTRING_WARNING_TRUNCATE The length of source exceeds the size of destination Increase the size of destination
