Description

IecStringCopy copies a string. It is assumed the size of destination is available when calling to protect from overflow. STRING variables, reference to STRINGs or literals are permitted inputs in IEC languages.

Unlike strncpy, IecStringCopy copies all or part (if longer than size) of source and ensures a null terminating string. Destination is not padded with zeros if source is shorter than size - only the null terminating character is copied.

The function immediately returns without processing if any string is NULL or size is zero.

Source and destination should not overlap.
