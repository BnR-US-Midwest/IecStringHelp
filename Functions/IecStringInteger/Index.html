IecStringInteger

Pad a whole number with zeros or spaces.

The function returns a status code, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN value DINT Signed integer
IN width USINT Minimum character width of destination string 1 to 10
IN space USINT Pad with zeros if zero and spaces otherwise
