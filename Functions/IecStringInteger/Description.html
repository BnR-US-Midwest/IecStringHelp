IecStringInteger Examples

IEC Structured Text

    VAR
        MyInteger : INT := 123;
        PaddedInteger : STRING[80];
    END_VAR
  
    PROGRAM _CYCLIC
        IecStringInteger(PaddedInteger, SIZEOF(PaddedInteger)MyInteger, 6, FALSE);
    END_PROGRAM

The value of PaddedInteger is '000123'.
