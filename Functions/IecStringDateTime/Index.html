IecStringDateTime

Format date and/or time.

The function returns a status number, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN value DTStructure Date time value
IN format STRING Format string

Delimeters

-, _, ., ,, /, :, ;, (, ), [, ]

Tokens

Token Component Example(s)
yy Year 01, 02
yyyy Year 2001, 2002
M Month 1, 12
MM Month 01, 12
MMM Month Jan, Dec
MMMM Month January, December
d Day 1, 31
dd Day 01, 31
ddd Weekday M, Tu
dddd Weekday Monday, Tuesday
H 24 hour 1, 14
HH 24 hour 01, 14
h 12 hour 1, 2
hh 12 hour 01, 02
m Minute 1, 59
mm Minute 01, 59
s Second 1, 59
ss Second 01, 59
tt Morning/afternoon AM, PM
ff Centiseconds 01, 99
fff Milliseconds 011, 999

Possible Errors

Constant Description Remedy
IECSTRING_ERROR_NULL Destination string is null (the memory address is zero) Destination must have a valid memory address
IECSTRING_ERROR_SIZE_ZERO Size is zero Size must be non-zero
IECSTRING_ERROR_OVERLAP The size of destination overlaps with the length of format Destination's memory location must differ from format's
IECSTRING_WARNING_TRUNCATE The resulting string length exceeds the size of destination Increase the size of destination
