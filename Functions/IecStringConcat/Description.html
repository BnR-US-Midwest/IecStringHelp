Description

IecStringConcat appends to (concatenates) a string. It is assumed the size of destination is available when calling to protect from overflow. STRING variables, reference to STRINGs or literals are permitted inputs in IEC languages.

IecStringConcat behaves like strncat if num (number of characters appended in strncat) is equal to size minus one.

The function immediately returns without processing if any string is NULL or size is zero.

Source and destination should not overlap.
