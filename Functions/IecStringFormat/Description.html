IecStringFormat Examples

IEC Structured Text

    VAR
        FromattedString : STRING[80];
        FormatValues : IecStringFormatType;
    END_VAR
  
    PROGRAM _INIT
        FormatValues.b[0] := TRUE;
        FormatValues.f[0] := 12.34;
        FormatValues.i[0] := 33;
        FormatValues.s[0] := 'All Clear';
    END_PROGRAM
  
    PROGRAM _CYCLIC
        IecStringFormat(FromattedString, SIZEOF(FromattedString)'Error: %b, Temp: %f C, Step: %i, Message: %s', FormatValues);
    END_PROGRAM

The value of FormattedString is 'Error: TRUE, Temp: 12.34 C, Step: 33, Message: All Clear'.
