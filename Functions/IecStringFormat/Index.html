IecStringFormat

Format string with runtime data.

The function returns a status code, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN source STRING Source string
IN values IecStringFormatType Format values

Format Specifiers

Similar to printf, the resulting string can be formatted using specifiers in the source string.

Specifier Description
%b Place the next boolean (BOOL) value from the values array
%f, %r Place the next floating point (REAL) value from the values array
%i, %d Place the next signed integer (DINT) value from the values array
%i, %d Place the next string (STRING[IECSTRING_FORMAT_LENGTH]) value from the values array

Possible Errors

Constant Description Remedy
IECSTRING_ERROR_NULL Destination and/or source string is null (the memory address is zero) Destination and source must have a valid memory address
IECSTRING_ERROR_SIZE_ZERO Size is zero Size must be non-zero
IECSTRING_ERROR_OVERLAP The size of destination overlaps with the length of source Destination's memory location must differ from source's
IECSTRING_WARNING_TRUNCATE The resulting string length exceeds the size of destination Increase the size of destination
