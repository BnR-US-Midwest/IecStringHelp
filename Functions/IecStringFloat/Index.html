IecStringFloat

Convert float to string.

The function returns a status number, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN value REAl Floating point value
IN width USINT Minimum character width
IN precision USINT Decimal precision
IN flags USINT Format flags

Supported Flags

Flag Description
IECSTRING_FLAG_E_NOTATION Use scientific notation

Possible Errors

Constant Description Remedy
IECSTRING_ERROR_NULL Destination string is null (the memory address is zero) Destination must have a valid memory address
IECSTRING_ERROR_SIZE_ZERO Size is zero Size must be non-zero
IECSTRING_ERROR_SIZE_INVALID The size is insufficient to hold the minimum character width of the number Increase size to support the character length of the number
