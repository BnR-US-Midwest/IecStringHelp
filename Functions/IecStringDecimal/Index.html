IecStringDecimal

Convert integer to string as decimal.

The function returns a status code, zero if successful.

I/O Parameter Data Type Description
IN destination STRING Destination string
IN size UDINT Size of destination
IN value DINT Signed integer
IN width USINT Minimum character width
IN flags USINT Format flags

Supported Flags

Flag Description
IECSTRING_FLAG_SIGN Preceed number with '+' or '-' sign
IECSTRING_FLAG_SPACES Left-pad number with spaces instead of zeros to fill width

Possible Errors

Constant Description Remedy
IECSTRING_ERROR_NULL Destination string is null (the memory address is zero) Destination must have a valid memory address
IECSTRING_ERROR_SIZE_ZERO Size is zero Size must be non-zero
IECSTRING_ERROR_SIZE_INVALID The size is insufficient to hold the minimum character width of the number Increase size to support the character length of the number or minimum width
IECSTRING_WARNING_TRUNCATE The resulting string length exceeds the size of destination Increase the size of destination
